# Team memory has moved → Obsidian vault

The team-shared project memory that used to live here as flat `.md` files is now a
canonical **Obsidian vault** at the repo root:

    /Obsidian-MangaDock/

Open that folder as an Obsidian vault (or just read the `.md` files) and start from
**`Home.md`** — the Map-of-Content index. Each memory is one note with `type` /
`description` frontmatter and `[[wikilinks]]` to related notes; Graph View shows the
relationships and surfaces gaps (unresolved links = memories worth writing).

Why the move: a single source of truth (no `.claude/memory` ↔ vault drift) plus a
navigable graph the agent can traverse instead of a flat pile of files.

Write new team memories as notes in `/Obsidian-MangaDock/` and add a line to `Home.md`.
