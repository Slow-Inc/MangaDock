## Suggested Usage in Paper

- Phase 1 ใช้สำหรับบท Project Planning
- Phase 2 ใช้สำหรับบท Requirement Analysis และ System Design
- Phase 3 ใช้สำหรับบท Prototype และ User Interaction Design
- Phase 4 ใช้สำหรับบท Internal Process, Documentation และ Version Control
- Phase 5 ใช้สำหรับบท Testing และ UAT
- Phase 6 ใช้สำหรับบท Deployment, Installation และ Operation
- Phase 7 ใช้สำหรับบท Evaluation, User Satisfaction และ Process Quality Evidence
